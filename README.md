# Installation:
```git clone https://github.com/p4intcantaim/p4ints-Python-Emporium```

# Instructions:
- Go to the directory of "The_Emporium.py" in a terminal.
- Type ```python3 The_Emporium.py``` in your terminal.
- Thats it, you are done!
